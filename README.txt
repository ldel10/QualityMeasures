How To Run:
Prerequisite: You will need Java installed to run this application. You can run "java -version" to see if it is installed.
1) Unzip folder
2) Add all data files to /data folder at application level
3) Add all schema file to /schema folder at application level
4) In the main project root level, there is a file called Nava.jar, configured for you to run on the command line
5) From the command line, run "java -jar Nava.jar"
